I had A LOT of problems trying to size svgs, it is like a DEMON to behave on an html document. Fortunatelly, I discovered a way to do this: use transform: scaleY(.x). Theres a problem, it will work for sizing, but at the same time, it will be floating above where we want to place it. To solve this problem, we can use the transform-origin prop. It sets the point where the transformation will be target. Set it to be top or bottom...
